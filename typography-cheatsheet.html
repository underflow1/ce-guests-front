Шпаргалка по типографике

Проверка загрузки стилей…

Стили собираются из базы text и модификаторов.
Размеров всего 3: base (13px), up (15px), down (12px). Операторский интерфейс тут не рассматриваем.

Шрифт (стек): Ubuntu, Segoe UI, Tahoma, Geneva, Verdana, sans-serif — берётся из --font-base в :root.

Общий стиль интерфейса (гайд)

Характер
Строго, спокойно, без “игрушечности”. Минимум декоративных эффектов.
Углы
Преимущественно прямые, но допустим лёгкий радиус (4–6px) для панелей/полей.
Примеры
Квадратный блок
radius: 0
Текст внутри блока
Скруглённый блок
radius: 6px
Текст внутри блока
Кнопки
Кнопки — простые, монохромные, 2 уровня: обычная (outline) и primary (заливка).
Поля ввода
Поля — лёгкая рамка, без “тени”, одинаковая высота/паддинги.
Отступы
Плотно, но не тесно. Если сомневаемся — уменьшаем на шаг.
Акценты
Акцент — через жирность и Up , а не через новые размеры.
Цвет
Primary / Muted / Subtle — для иерархии текста, а статусы — отдельными семантическими цветами.
Монохром
Стараться делать интерфейс максимально монохромным, “бумажным”. Цвет — только по делу (статусы/ошибки).

7 базовых “стилей” (как комбинации)

Base
Текст базовый (13px) — используется “по умолчанию”
Base Strong
Базовый жирный — акцент без увеличения размера
Base Light
Базовый тонкий — когда нужно “легче”, но не бледнее
Base Italic
Базовый курсив — например, ответственное лицо / примечания
Up
Увеличенный (15px) — единственный шаг “крупнее”
Up Strong (Headline)
Заголовок / важное место — заголовки панелей, секции
Down
Уменьшенный (12px) — метаданные, подписи, “пилюли”

Тональности текста (иерархия важности)

Тональность — это не “статус”, а визуальная иерархия: основной текст → вторичный → третичный.

Primary
Основной текст (по умолчанию)
Muted
Вторичный текст (подписи, даты, пояснения)
Subtle
Третичный текст (ещё тише: коды, второстепенные детали)

Декорации (не тональность и не семантический цвет)

Это “состояния” текста (например, выполнено/отменено). Их можно комбинировать с тональностью.

Strike
Зачёркнутый вторичный текст — пример для “отменено / неактуально”

Семантические цвета (это не “тональности”)

Это цвета статусов/смыслов (ошибка/успех/предупреждение/инфо). Их не считаем частью типографики.
