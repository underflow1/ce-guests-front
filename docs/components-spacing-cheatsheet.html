Шпаргалка — стиль и компоненты

Типографика, отступы (gap / space), кнопки, поля, цвета. Единый гайд по стилю интерфейса.

Типографика

База text + модификаторы. Размеры: --text-size-down (12px), --text-size-base (13px), --text-size-up (15px). Шрифт --font-base — Ubuntu, Segoe UI, Tahoma…

Кнопки

В приложении есть ещё (не в шпаргалке)

Элементы из styles.css / настроек, которые можно добавить при необходимости:

  • Форма: .form, .form__field, .form__label, .form__control, .form__checkbox, .form-section, .form__submit-row, .form__bottom-actions
  • Кнопки: .button--danger, .button--success (семантика)
  • Иконка-кнопка: .icon-action-button, --primary, --success, --danger
  • Навбар настроек: .layout__item, .layout__item--active (пункт меню + активное состояние)
  • Блоки: .section-group, .section-group__title, .section-card
  • Поля: .input--wide, .input--grow
  • Обратная связь: .toast, .toast--error/success/warning/info, .error-message
  • Таблица: .table__status-active, .table__status-inactive
